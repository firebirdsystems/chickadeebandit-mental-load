# Mental Load

A fair-play tracker for couples, based on the *Fair Play* model. Chores apps track task
*execution* — Mental Load tracks *ownership* of the invisible domains: planning meals,
remembering birthdays, booking the kids' checkups, staying on top of bills.

- **Name the domains** — each card is a whole job (conception → planning → execution), not a one-off task.
- **See the balance** — a live bar of who holds what, so the imbalance is visible instead of felt.
- **Reassign in one tap** — cycle a domain's owner between the two of you.
- **Rebalance check-ins** — periodically talk it through and log what you agreed to shift.

Private to the two partners: data is `couple_scoped`, so only you and the partner you've each
selected can see the board. Pairing is reciprocal — you both pick each other before anything appears.

## Quick start

```bash
npm run dev     # http://localhost:3001
npm run build   # produces dist/bundle.json
npm test
```
